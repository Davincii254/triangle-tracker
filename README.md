# triangle-tracker


This is a website that allows the userr to enter the lenght of all three sides of a triangle and produe the type of the triangle.

Technologies Used HTML & CSS (Bootstrap) JavaScript Animate CSS Requirements JSFiddle or JavaScript Console

Setup Instructions and Installation Clone this repository to a location in your file system. 'git clone https://github.com/Davincii254/triangle-tracker.git Open terminal command line then navigate to the root folder of the application. cd triangle-tracker Open index.html on your Browser.
